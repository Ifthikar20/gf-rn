# Audio Assets

Place your MP3 files in this directory for local audio playback.

## Sample Audio Files

You can add meditation, relaxation, or ambient sound MP3 files here:

- `rain-sounds.mp3` - Rain ambient sounds
- `ocean-waves.mp3` - Ocean wave sounds
- `forest-ambience.mp3` - Forest nature sounds
- `meditation-bells.mp3` - Meditation bell sounds
- `white-noise.mp3` - White noise for focus

## Free Audio Resources

To get free audio files, you can download from:
- [Pixabay Audio](https://pixabay.com/music/)
- [FreeSound](https://freesound.org/)
- [YouTube Audio Library](https://www.youtube.com/audiolibrary)

## Usage

Audio files can be loaded in the app using:
```javascript
const sound = require('./assets/audio/rain-sounds.mp3');
```
